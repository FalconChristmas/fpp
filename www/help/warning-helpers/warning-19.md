# FPP Scheduler Is Disabled
This player has the scheduler turned off, so scheduled playlists will not run automatically.

1. If this is intentional (manual control only), no action is needed.
2. To re-enable, set **Disable Scheduler** to off on the [System](../../settings.php#settings-system) settings page.

Note: the **Disable Scheduler** setting is only shown when **User Interface Level** (near the top of the System page) is set to **Advanced** or higher. In **Basic** mode it is hidden, so switch to **Advanced** first.
