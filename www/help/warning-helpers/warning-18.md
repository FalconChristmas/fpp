# FSEQ Data Block Not Available - Slow Storage
FPP needed the next block of the sequence file but it had not finished loading from storage, indicating the SD card or disk is too slow.

1. Use a high-quality, high-speed SD card (or USB/SSD storage) and re-copy sequences after replacing it.
2. Re-render sequences with compression in xLights to reduce the data read per frame.
3. A failing SD card also causes this — replace if errors persist.
