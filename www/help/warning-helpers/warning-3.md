# Log Level Set to Debug
One of your logging settings is at the **Debug** level. This logs much more than normal, which can hamper performance and use extra space on the SD card during a running show.

1. **Debug** logging is useful while diagnosing a problem, but should not be left on for normal running.
2. When you are done troubleshooting, set the log level back to **Warn** or **Info** on the [Logging settings](../../settings.php#settings-logs) page.
