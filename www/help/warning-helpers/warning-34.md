# Playlist Image Could Not Be Displayed
A playlist "image" step could not show its image on a Pixel Overlay Model.

- **Missing image path** — the step has no image file configured.
- **Could not read image file** — the file is missing, corrupt, or an unsupported format.

1. Open the [Playlists](../../playlists.php) editor and check the image step's file path.
2. Confirm the image is uploaded via the [File Manager](../../filemanager.php) and the name matches exactly (case-sensitive).
3. Use a standard image format (PNG/JPG/GIF/BMP); re-save the file if it may be corrupt.

This warning clears automatically after a short time once the problem stops recurring.
